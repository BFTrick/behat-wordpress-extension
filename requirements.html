Requirements

  • A WordPress site, running version 4.7+.
  • PHP 5.4+, with the cURL extension enabled.
  • Composer.
  • Selenium ("Standalone Server" version). 
    java -jar selenium-server-standalone-3.0.1.jar

Driver-specific requirements

The WP-CLI driver (the default) requires WP-CLI, installed globally:

  curl -o wp https://raw.githubusercontent.com/wp-cli/builds/gh-pages/phar/wp-cli.phar
  chmod +x wp

The executable must be named wp and be within your system's $PATH.
