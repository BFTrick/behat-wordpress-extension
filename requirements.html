System Requirements

  • A WordPress site, running version 4.7+.
  • WP-CLI installed globally.
    • Thi binary must be named wp and be within your system's $PATH.
  • PHP 5.4+, with the cURL extension.
  • Composer.
  • Selenium.
    • Download the Standalone Server, place the file wherever you want, and run with the following command: 
      $ java -jar selenium-server-standalone-3.0.1.jar
